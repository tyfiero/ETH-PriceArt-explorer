Help

Welcome to ETH/USD PriceArt!

  • For the best experience, use chrome on desktop.
  • Click and drag on the canvas to rotate the model, right click and drag to move the model, and click the mousewheel to reset the model view.
    • file_download Downloads png image. (Desktop only)
    • home Resets model back to the default state.
    • nights_stay Toggles dark mode.
    • star Toggles the background stars.
    • layers Turns on the data labels and ETH logo.
    • fast_forward Toggles the model animation settings.
    • casino Randomizes all values.
    • add_photo_alternate Allows you to upload a custom background.
      Square images with a small file size work best.
      (By using this feature you agree to only use images that you own the rights to.)
